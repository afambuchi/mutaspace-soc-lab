# Custom PC Build Day Checklist

## Purpose

This checklist guides the physical assembly of the official MutaSpace SOC Lab host.

## Learning Goals

By completing this stage, I should understand:

1. How the motherboard, CPU, RAM, storage, cooler, PSU, and case fit together.
2. Why careful assembly matters before installing Proxmox.
3. How to validate that the system powers on successfully.
4. Why BIOS settings matter before installing a hypervisor.

## Before Starting

- [ ] Clear workspace
- [ ] Gather screwdriver
- [ ] Open all parts carefully
- [ ] Keep manuals nearby
- [ ] Avoid carpet if possible
- [ ] Ground yourself before touching components
- [ ] Confirm all parts arrived undamaged

## Assembly Order

- [ ] Install CPU into motherboard
- [ ] Install RAM into correct slots
- [ ] Install NVMe SSD
- [ ] Prepare case
- [ ] Install motherboard into case
- [ ] Install PSU
- [ ] Connect motherboard power cables
- [ ] Mount 360mm radiator and fans
- [ ] Connect cooler pump and fans
- [ ] Connect front panel connectors
- [ ] Connect USB and audio headers
- [ ] Connect case fans
- [ ] Cable manage
- [ ] First power-on test

## First Boot Validation

- [ ] System powers on
- [ ] Fans spin
- [ ] RGB/fan headers work if connected
- [ ] BIOS loads
- [ ] CPU detected correctly
- [ ] RAM detected correctly
- [ ] NVMe detected correctly
- [ ] CPU temperature looks reasonable in BIOS
