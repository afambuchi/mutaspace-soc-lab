# Parts List

## Purpose
This document details the parts purchased for the custom pc build. The hardware determines what the lab can support. The CPU gives compute power, RAM determines how many VMs can run, NVMe storage holds active VMs, cooling projects long-running workloads, and the PSU provides stable power.

## Parts Confirmed so far

CPU: AMD Ryzen 9 7900X
PSU: MSI MAG A850GL PCIE5 850W
RAM: Silicon Power DDR5 64GB, 2 x 32GB, 6000MT/s
Case: CORSAIR 4000D RS ARGB
Cooler: CORSAIR Nautilus 360 RS ARGB
Storage: Silicon Power 2TB UD90 NVMe Gen4
